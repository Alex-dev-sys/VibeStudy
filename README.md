## 🚀 Быстрый старт

### ⚡ **Вариант 1: Запуск за 2 минуты**

Для тех, кто хочет **попробовать прямо сейчас:**

```bash
# 1️⃣ Клонируй репозиторий
git clone https://github.com/Alex-dev-sys/VibeStudy.git
cd VibeStudy

# 2️⃣ Установи зависимости
npm install

# 3️⃣ Создай .env.local с минимальными настройками
cat > .env.local << 'EOF'
HF_TOKEN=твой_токен_от_gptlama
HF_API_BASE_URL=https://api.gptlama.ru/v1
HF_MODEL=gpt-4o-mini
EOF

# 4️⃣ Запусти!
npm run dev
```

🎉 **Готово!** Открой http://localhost:3000 и начинай учиться!

<details>
<summary>📍 <strong>Где взять HF_TOKEN?</strong></summary>

1. Зарегистрируйся на [gptlama.ru](https://gptlama.ru/)
2. Получи API токен в личном кабинете
3. Вставь в `.env.local`

**Это обязательно** для работы AI-функций (генерация задач, проверка кода, подсказки)

</details>

### 🌟 **Вариант 2: Полная настройка (с синхронизацией)**

Для **максимального опыта** с облаком и Telegram:

<details>
<summary>📋 <strong>Развернуть полную инструкцию</strong></summary>

```bash
# 1️⃣ Клонируй и установи
git clone https://github.com/Alex-dev-sys/VibeStudy.git
cd VibeStudy
npm install

# 2️⃣ Создай полный .env.local
cat > .env.local << 'EOF'
# 🤖 AI (обязательно для работы)
HF_TOKEN=твой_токен_gptlama
HF_API_BASE_URL=https://api.gptlama.ru/v1
HF_MODEL=gpt-4o-mini

# ☁️ Supabase (для синхронизации)
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...
SUPABASE_SERVICE_ROLE_KEY=eyJxxx...

# 📱 Telegram Bot (для уведомлений)
TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
CRON_SECRET=твой_случайный_секрет

# 🌐 Production URL (для webhooks)
NEXT_PUBLIC_APP_URL=https://your-domain.com
EOF

# 3️⃣ Настрой Supabase
# - Создай проект на supabase.com
# - Выполни SQL из supabase/schema.sql
# - Настрой Google OAuth в Authentication

# 4️⃣ Настрой Telegram (опционально)
npm run bot:webhook

# 5️⃣ Запусти с ботом
npm run dev
```
