# Product Overview

VibeStudy is a 90-day programming education platform designed to take learners from absolute beginner to junior developer level.

## Core Features

- **Structured Learning**: 90-day curriculum covering 7 programming languages (Python, JavaScript, TypeScript, Java, C++, C#, Go)
- **AI-Powered Content**: Dynamic theory and task generation using Hugging Face API
- **Interactive Code Editor**: Monaco Editor with syntax highlighting for all supported languages
- **Gamification**: 21 achievements across 4 categories, streak tracking, and progress visualization
- **Authentication**: Guest mode (browser-only), Google OAuth, and email magic links via Supabase
- **Cloud Sync**: Optional Supabase integration for cross-device progress synchronization
- **Telegram Bot**: Reminders, progress tracking, and motivational messages
- **Analytics**: Detailed statistics, activity calendar, and progress graphs
- **Playground**: Code sandbox for experimentation across all languages
- **Multilingual**: Russian (default) and English interfaces

## User Journey

1. Choose a programming language
2. Progress through 90 days of structured content
3. Complete theory, tasks, and daily challenges
4. Track achievements and maintain learning streaks
5. Use playground for experimentation
6. Sync progress across devices (optional)

## Guest Mode

The platform fully supports guest mode where users can learn without authentication. Progress is stored in browser localStorage with optional cloud sync when users authenticate.
